# Laptop Price Predictor 💻

This is a Machine Learning web application that predicts laptop prices based on user input.

Project Overview:
This project is a Machine Learning solution designed to estimate the market price of laptops. It takes various hardware specifications as input—such as RAM, CPU, GPU, and Weight—and predicts the price in Sri Lankan Rupees (LKR). The project aims to help users get a quick and accurate idea of a laptop's value before purchasing.

## How to use:
1. Enter the RAM, Weight, and other specifications.
2. Click on 'Predict Price'.
3. The estimated price in LKR will be displayed.

## Technologies Used:
* Python (Machine Learning)
* Flask (Web Framework)
* Scikit-learn (Random Forest Model)
* HTML/CSS (UI Design)
